# Dogs

Rede social para cachorros baseada no projeto Dogs da Origamid.

## Stack Atual

- React 19
- React Router 6
- Vite 6
- Yarn Classic

## Requisitos

- Node 18 ou superior
- Yarn 1.x

React Router 7 e Vite 8 exigem Node 20+, então esta modernizacao mantem as ultimas versões compativeis com o ambiente atual.

## Scripts

Instalar dependencias:

```bash
yarn install
```

Rodar em desenvolvimento:

```bash
yarn dev
```

Build de producao:

```bash
yarn build
```

Preview do build:

```bash
yarn preview
```

Testes:

```bash
yarn test
```

No momento, `yarn test` e um placeholder. A configuracao real de testes sera adicionada em uma issue propria.
