## Contexto

`FeedModal` esta vazio. O app precisa permitir abrir uma foto para ver detalhes, seguindo o comportamento esperado do Dogs.

## Objetivo

Implementar modal de foto com detalhes e fechamento acessivel.

## Escopo

- Controlar foto selecionada no `Feed`.
- Abrir modal ao clicar em item do feed.
- Buscar ou receber dados necessarios da foto.
- Fechar ao clicar fora, apertar Escape ou acionar botao.
- Garantir foco e acessibilidade basica do modal.

## Criterios De Aceite

- Clique em uma foto abre modal.
- Modal fecha sem recarregar a pagina.
- Layout funciona em desktop e mobile.
- Elementos interativos tem labels acessiveis.
