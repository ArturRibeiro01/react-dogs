## Contexto

O README ainda e o padrao do Create React App. Para portfolio, ele precisa explicar o produto, stack, decisoes tecnicas, como rodar e o que foi modernizado.

## Objetivo

Transformar o README em uma pagina de apresentacao profissional do projeto.

## Escopo

- Descrever o que e o Dogs.
- Listar features principais.
- Documentar stack e arquitetura em alto nivel.
- Adicionar instrucoes de instalacao, ambiente e scripts.
- Adicionar screenshots ou GIF.
- Adicionar secoes de melhorias futuras e aprendizados.

## Criterios De Aceite

- README nao contem mais texto generico do CRA.
- Uma pessoa consegue rodar o projeto seguindo o README.
- O README comunica evolucao de junior para senior sem exagero.
